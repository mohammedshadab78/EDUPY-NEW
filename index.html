How it works

Three steps to get going.

A simple workflow designed to build real understanding, not just copy-paste habits.

1
Read the notes

Start with the Notes page. Follow modules in order — basics, input/output, conditions, loops, functions, strings, lists, dicts.

2
Practice in the editor

Re-write examples, modify them, break them on purpose. Watching the output and AST teaches you far more than just reading.

3
Solve assessments

Attempt questions level by level. Use hints only when stuck, then review solutions to pick up better patterns.

About EduPy

A project with a purpose.

Built by Mohammed Shadab to make Python approachable for everyone starting out.

Why this exists

Installing Python, configuring IDEs, and decoding cryptic error messages stops a lot of beginners before they even start. EduPy removes all of that friction so you can focus on learning to think like a programmer.

The goal isn't to replicate Python perfectly — it's to teach Python-style thinking: variables, conditions, loops, functions, data structures, and real problem solving in a safe sandbox built for experimentation and mistakes.

  • 100% browser-based — works on low-end devices.
  • Clean, distraction-free editor with helpful hints.
  • Roadmap-style notes from basics to advanced topics.
  • Assessments that push you to apply, not just read.

Where it's headed

EduPy works as a personal learning tool, a college lab companion, or a portfolio project that shows genuine interest in compilers and education technology.

Future ideas: deeper Python compatibility, richer problem sets, and integration with real Python interpreters for advanced learners.
